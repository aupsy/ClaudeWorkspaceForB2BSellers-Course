# Accounts

One folder per account. Copy `_template/` to get started.

## Naming Convention

Use the account's legal or commonly-known name, no spaces:
- `Acme-Corp/`
- `Sandvik-Coromant/`
- `Marsh-McLennan/`

## Files in Each Account Folder

| File | Purpose | Update frequency |
|------|---------|-----------------|
| `Account-Overview.md` | Who they are, stakeholders, deal history | When something material changes |
| `Engagement-Log.md` | Running log of every call, email, meeting | Within 24 hours of each interaction |
| `Open-Items.md` | Action items, commitments, open questions | After every call |

## Lifecycle

- **Active opportunity**: Full folder with all three files maintained
- **Prospect (not yet engaged)**: Just `Account-Overview.md` — enough for research and outreach
- **Closed-won**: Move notes to a `Win-Loss/Wins/[Account]-[Date].md` write-up; archive or keep folder
- **Closed-lost**: Write a loss retrospective in `Win-Loss/Losses/`; keep folder for future cycles

## Tips

- Log call notes immediately — memory degrades fast
- Include exact quotes from the customer when possible ("they said 'our biggest pain is...'")
- Name stakeholders with titles: "Jane Doe (VP Sales, Economic Buyer)"
- Flag competitors mentioned: "mentioned they're also evaluating Competitor X"
