# Monday Morning Workflow

Copy this task into Claude Cowork at the start of each week. Replace the placeholder account names and prospect list with your own. Run it while you prepare for your first call of the day.

---

## The Workflow Task

```
Task: Prepare my week for maximum selling time

Context:
- My name: [Your Name], [Title] at [Company]
- My ICP: [2 sentences]
- My most important deals this week: [Account 1], [Account 2], [Account 3]
- Prospects I'm outreaching to this week: [Prospect A], [Prospect B], [Prospect C], [Prospect D], [Prospect E]
- Unprocessed call transcripts from last week: [Account Name(s), or "none"]
- Today's date: [Date]

Steps — run in parallel where possible:

1. PIPELINE SCAN: Review my pipeline (paste contents of Course-Materials/sample-pipeline.md
   or read from CRM if connected). Flag any deals needing attention this week — specifically:
   - Close dates within 14 days
   - Deals with no activity in 10+ days
   - Deals stuck in the same stage for 21+ days
   Deliver as: Accounts/[today's date]-pipeline-flags.md

2. ACCOUNT BRIEFS: For each of my 3 most important deals, create a one-page account brief
   with latest news (last 30 days), inferred current state of the deal, and a suggested
   talk track for this week's interaction.
   Deliver as: Accounts/[Account Name]/brief-[today's date].md (one per account)

3. OUTREACH: For each prospect in my outreach list, draft a cold email referencing a
   specific, researched trigger event. If no trigger is findable, flag it rather than
   inventing one.
   Deliver as: Outreach/[Prospect]-[today's date].md (one per prospect)

4. CALL PROCESSING: If there are unprocessed call transcripts listed above, process each one:
   - Structured summary (pain, signals, objections, next steps)
   - Follow-up email draft
   - CRM update fields
   - One coaching note
   Deliver as: Accounts/[Account Name]/post-call-[date].md

5. DAILY PRIORITY LIST: Based on all of the above, create a prioritized action list for today.
   Top 5 items only — not a full to-do list. Format as: what to do, why it matters, when.
   Deliver as: Accounts/[today's date]-daily-priorities.md

Deliver each output as a separate file in the specified location. Flag any inputs that
are missing or unclear rather than making assumptions.
```

---

## Manual Equivalent vs. Cowork Time

| Task | Manual | Cowork |
|------|--------|--------|
| Pipeline review + flags | 20–30 min | ~1 min |
| 3 account briefs | 45–90 min | ~2 min |
| 5 prospect emails | 75–150 min | ~2 min |
| Call processing (per call) | 60–90 min | ~2 min |
| Priority list | 10–15 min | ~1 min |
| **Total** | **3.5–6+ hrs** | **~8 min** |

The recovered time is meant for calls, not for more tasks.

---

## Customizing the Workflow

**Add your own steps**: If you have a recurring Monday task (territory report, manager update, account list scrub), add it as a numbered step. Cowork can handle up to ~8–10 parallel steps before quality degrades.

**Reduce scope**: If 5 prospect emails is too many, drop to 3. The goal is a workflow you'll actually run every week, not the most ambitious one.

**Connect CRM**: When CRM is connected via MCP, replace Step 1's pipeline paste with: "Query CRM for all open opportunities. Identify..." — the output is live and current rather than based on pasted data.
