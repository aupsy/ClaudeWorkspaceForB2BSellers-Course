# Course Materials

Practice materials for in-class exercises. Use these when you don't have real data available — or as a supplement to understand what good outputs look like.

---

## Sample Transcripts (Module 4)

| File | Scenario | Use for |
|------|---------|---------|
| `sample-transcripts/discovery-call-enterprise.md` | 45-min discovery call with VP Sales at a 3,500-person manufacturing company. Competitor mentioned. Timing objection. Strong buying signal. | `/post-call Meridian` — the coaching note catches something the rep missed |
| `sample-transcripts/discovery-call-midmarket.md` | 30-min call with a SaaS CRO. No competitor. Clear urgency. Clean next step. | `/post-call Vantage` — good contrast to the enterprise call; shows what a well-structured call produces |

**How to use them:**
1. Create an account folder: `Accounts/Meridian/` (or `Accounts/Vantage/`)
2. Copy the transcript file into that folder
3. Run `/post-call Meridian` (or `/post-call Vantage`)

## After the Course

These materials are for practice only. Once you're using the workspace with real accounts:
- Replace sample materials with real call notes in `Accounts/[Your Account]/`
- Run `/post-call` on actual transcripts from Gong, Chorus, or Zoom
- Use `/pipeline-review` with live CRM data (or manual pipeline paste from your CRM export)
